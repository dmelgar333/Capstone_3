# Capstone 3: EmporiUm Sales Analysis using Power BI

## Project Overview
Analysis of East Region: New York & Connecticut sales data using Power BI.

## Territory
- Region: East
- States: New York & Connecticut
- Time Period: 2022–2025

## Visualizations
- Page 1: Sales Overview — sales trends over time, sales by category
- Page 2: Regional Highlights — sales by state, top-selling books

## Key Findings
- Technology & Accessories is the top category at $30M in sales
- New York drives 64.45% of East region revenue
- Store sales had consistent growth from $620K/month in 2022 to $1.8M by 2025
  

## Tools Used
Power BI Desktop, Power Query, DAX, GitHub
