# Capstone 3: EmporiUm Sales Analysis with Power BI

## Project Overview
Analysis of EmporiUm East Region: New York & Connecticut
sales data from 2022-2025 using Power BI.

- Page 2: Regional Highlights: sales by state, top-selling books
